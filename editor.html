💡 Quick JSON Schema Cheat-Sheet

1. Slide Cover (type: "title") 

{
  "type": "title",
  "title": "Slide Title",
  "subtitle": "Subtitle",
  "authors": "Tio Haidar Hanif\ntio@example.com",
  "notes": "Speaker notes here..."
}

2. Bullet Points (type: "content")

{
  "type": "content",
  "title": "Key Features",
  "bullets": [
    "Core Features:",
    "  ▪ Sub feature indentation",
    "  1. Step-by-step numbers"
  ],
  "notes": "Speaker notes here..."
}

3. Split Layout (type: "split")

Split the slide in two columns. Left column renders `bullets` array. Right column is determined by `rightType` (`table`, `chart`, or `metric`).

rightType: "table" 
"rightType": "table",
"table": {
  "headers": ["Col A", "Col B"],
  "rows": [
    ["Data 1", "Val X"],
    ["Data 2", "Val Y"]
  ]
}
rightType: "chart" 
"rightType": "chart",
"chart": {
  "type": "bar", // bar, line, pie
  "labels": ["Q1", "Q2", "Q3"],
  "values": [40, 60, 95]
}
rightType: "metric" 
"rightType": "metric",
"metric": {
  "value": "98%",
  "label": "Efficiency Score"
}
